## Installation

**Requirements:**

- Python 3.6.2 or more recent
- PIP
- [Pipenv](https://pypi.org/project/pipenv/)

Clone this repository and run `pipenv install`.

# Running the project

Invoke the following command:

```console
$ pipenv run python main.py
```
